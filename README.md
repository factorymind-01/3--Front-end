## Tech Stack

React.js, Typescript, Material-UI v5+, Apexcharts


## Run Locally

Clone the project

```bash
  git clone git@github.com:factorymind-01/3--Front-end.git
```

Go to the project directory

```bash
  cd 3--Front-end
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
